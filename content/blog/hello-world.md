## ブログ開設のお知らせ

株式会社団野ソフトウェアの技術ブログを開設しました。

このブログでは、日々の業務で得た技術的な知見や、クラウドインフラ・AI活用に関する情報を発信していきます。

### 発信予定のテーマ

- **クラウドインフラ**: AWS / GCP / Azure の設計・構築に関するノウハウ
- **Infrastructure as Code**: Terraform や Bicep を使ったインフラのコード管理
- **AI活用**: Claude Code や Codex を使った開発の実践例
- **技術トピック**: 日々の開発で気づいたことや学んだこと

### 更新頻度

不定期ですが、月に数本のペースで記事を公開していく予定です。

ぜひご覧いただければ幸いです。
