## プロジェクト概要

株式会社団野ソフトウェアのコーポレートサイト。Vite + React + TypeScript。Vercelでホスティング。

**パブリックリポジトリのため、機密値（APIキー、トークン等）は絶対にコードに含めないこと。**

## コマンド

- `npm run dev` — 開発サーバー起動
- `npm run build` — プロダクションビルド
- `npm run lint` — ESLint
